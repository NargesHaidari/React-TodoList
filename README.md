# React-TodoList

A simple TodoList application built with React, Vite, and Tailwind CSS.
This project is created for practicing React fundamentals like component structure and UI design.

## Current status
- Todo UI layout completed
- Component-based structure implemented
- Basic input field added (no functionality yet)

## Planned features
- Add new task functionality
- Delete task functionality
- Mark tasks as completed
- Store tasks in localStorage

## Tech stack
- React
- Vite
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
