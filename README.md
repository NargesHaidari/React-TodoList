# React-TodoList

A simple TodoList application built with React, Vite, Tailwind CSS, and UUID for unique task identification.
This project is created for practicing React fundamentals like component structure, state management, and UI design.

## Current Status

* TodoList UI completed
* Component-based structure implemented
* Task input field added
* Add new task functionality implemented
* Unique task IDs generated using uuidv4

## Planned Features

* Delete task functionality
* Mark tasks as completed
* Edit existing tasks

## Tech Stack

* React
* Vite
* Tailwind CSS
* UUID

## Installation

```bash
npm install
npm run dev
```
